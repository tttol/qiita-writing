# はじめに
普段使いのエディタをVSCodeからNeovimに移行してから、ターミナル上で生活する時間が増えました。それに伴ってターミナルアプリも見直すようになり、色々試した結果最近ではweztermを使うようになりました。（ターミナル.app → iTerm2 → wezterm（今ここ））


weztermは設定をかなり自由にカスタマイズできるのが特徴です。自分好みにカスタマイズした私の設定ファイルをここで紹介しようと思います。
