# GitHub
### ZENIGAME
https://github.com/tttol/zenigame-gen2

家計管理アプリです。立替精算の計算を楽にしてくれます。
家族専用に閉じたアプリなので、わりと好き勝手な実装をしています。

### amplify-passwordless-auth
https://github.com/tttol/amplify-passwordless-auth

Amplifyで作成したアプリケーションにパスワードレス認証を適用したサンプルです。

### nibble-amplify
https://github.com/tttol/nibble-amplify

Amplifyで気になった機能やアプデ等を検証するためのリポジトリです。

# さいごに

2025年も引き続き、アウトプットに精を出していきます！
